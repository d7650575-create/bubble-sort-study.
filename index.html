АЛГОРИТМ СОРТИРОВКИ ПУЗЫРЬКОМ

Суть метода:

Сортировка пузырьком (Bubble Sort) — это базовый алгоритм упорядочивания элементов. Его название обусловлено тем, что наибольшие значения постепенно «всплывают» в конец списка, подобно пузырькам воздуха в воде.

Механика работы:

Алгоритм совершает несколько проходов по массиву. На каждом этапе сравниваются два соседних числа. Если порядок нарушен (левое больше правого), они меняются местами. Этот процесс повторяется до тех пор, пока весь массив не станет упорядоченным.

Где это применяется:

Несмотря на простоту, алгоритм крайне важен для обучения. Он демонстрирует работу вложенных циклов и условных переходов. В промышленной разработке он используется для обучения ИИ или работы с уже почти отсортированными данными.

Анализ сложности:

Временная сложность составляет O(n^2). Это означает квадратичную зависимость: при увеличении количества данных в 3 раза, время обработки возрастет в 9 раз. Это делает алгоритм неэффективным для огромных массивов данных (Big Data).

ПРОГРАММИРОВАНИЕ И IDLE SHELL

from random import randint # Подключаем инструмент для случайных чисел
n = 20 # Размер нашего списка
num = [] # Создаем пустую "коробку" для чисел
for i in range(n): # Повторяем n раз
  num.append(randint(0, 100)) # Кладем случайное число от 0 до 100
print(num) # Показываем список до сортировки

for i in range(n - 1): # Проходим по списку n-1 раз
  for j in range(n - i - 1): # Сравниваем пары чисел
    if num[j] > num[j + 1]: # Если левое больше правого...
      num[j], num[j+1] = num[j+1], num[j] # ...меняем их местами

print(num) # Показываем результат
