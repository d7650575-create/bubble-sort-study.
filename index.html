Алгоритм сортировки пузырьком

Суть метода: Сортировка пузырьком (Bubble Sort) — это базовый алгоритм упорядочивания элементов. Его название обусловлено тем, что наибольшие значения постепенно «всплывают» в конец списка, подобно пузырькам воздуха в воде.

Механика работы: Алгоритм совершает несколько проходов по массиву. На каждом этапе сравниваются два соседних числа. Если порядок нарушен (левое больше правого), они меняются местами. Этот процесс повторяется до тех пор, пока весь массив не станет упорядоченным.

Где это применяется: Несмотря на простоту, алгоритм крайне важен для обучения. Он демонстрирует работу вложенных циклов и условных переходов. В промышленной разработке он используется для обучения ИИ или работы с уже почти отсортированными данными.

Анализ сложности: Временная сложность составляет $O(n^2)$. Это означает квадратичную зависимость: при увеличении количества данных в 3 раза, время обработки возрастет в 9 раз. Это делает алгоритм неэффективным для огромных массивов данных (Big Data).

Программирование на языке Python

Разбор программы шаг за шагом:

def bubble_sort(my_list): # Создаем функцию, которая принимает список чисел
    n = len(my_list) # Определяем длину массива (количество элементов)
    for i in range(n): # Внешний цикл: отвечает за количество проходов
        for j in range(0, n - i - 1): # Внутренний цикл: сравнение пар
            if my_list[j] > my_list[j+1]: # Условие: если левый элемент больше правого
                my_list[j], my_list[j+1] = my_list[j+1], my_list[j] # Меняем их местами

numbers = [5, 2, 9] # Вводим сам массив чисел для обработки
bubble_sort(numbers) # Вызываем нашу функцию для этого массива
